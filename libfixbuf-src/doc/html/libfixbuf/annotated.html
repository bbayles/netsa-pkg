Data Structures
Here are the data structures with brief descriptions:
fbBasicList_stA basic list element in a template which structure represents a basic list on the internal side, basic lists in an IPFIX Message must be represented by this structure within the application record
fbConnSpec_stConnection specifier
fbInfoElement_stA single IPFIX Information Element definition
fbInfoElementOptRec_stThe corresponding struct to the Information Element Type Options Template
fbInfoElementSpec_stA single IPFIX Information Element specification
fbInfoModelIter_stAn iterator over the information elements in an information model
fbListenerEntry_stListenerEntry's make up a listener group as a linked list
fbListenerGroupResult_stListenerGroupResult's contain the listener who's listening socket got a new connection
fbSubTemplateList_stStructure used to hold information of a sub template list
fbSubTemplateMultiList_stMultilists just contain the semantic to describe the sub lists, the number of sub lists, and a pointer to the first entry
fbSubTemplateMultiListEntry_stEntries contain the same type of information at SubTemplateLists: template ID and template pointers to describe the data the number of data elements and the data pointer and data length
fbTemplate_stAn IPFIX template or options template structure
fbTemplateOptRec_stTemplate metadata options record structure
fbUDPConnSpec_stAn UDP Connection specifier
fbVarfield_stA variable-length field value
